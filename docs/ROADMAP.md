# Дорожная карта SG Client Android

## 001 — основа приложения

- Compose UI;
- темы;
- импорт и классификация профилей;
- VpnService;
- unit-тесты;
- debug APK через Actions.

## 002 — первый рабочий протокол

- безопасное хранилище профилей;
- интеграция выбранного движка;
- реальное подключение и отключение;
- foreground-уведомление;
- диагностика процесса и TUN.

## 003 — экосистема SG

- VLESS из SG-Panel;
- Hysteria2 из SG-Panel;
- AmneziaWG из SG-AWG-Panel;
- единая модель профилей.

## Первый публичный релиз

Публикация GitHub Release начнётся только после того, как APK:

- устанавливается;
- импортирует профиль;
- реально подключает VPN;
- реально отключает VPN;
- показывает честный статус;
- содержит работающий журнал и диагностику.
