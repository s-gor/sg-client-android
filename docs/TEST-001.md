# Проверка SG-CLIENT-ANDROID-001

## 1. Установка

Скачайте artifact `SG-CLIENT-ANDROID-001` из последнего успешного запуска GitHub Actions и установите APK на Android 8.0 или новее.

На телефоне может потребоваться разрешить установку приложений из используемого браузера или файлового менеджера.

## 2. Запуск

После запуска должны быть видны:

- название SG Client;
- экран «Подключение»;
- нижнее меню;
- честный статус «Отключено»;
- пояснение, что сетевые движки ещё не подключены.

## 3. Темы

Откройте «Настройки» и последовательно выберите:

- Системная;
- Светлая;
- Тёмная.

Тема должна меняться сразу.

## 4. Импорт VLESS

Откройте «Профили», нажмите «Импортировать профиль» и вставьте тестовую VLESS-ссылку.

Проверяется только распознавание формата. Секретные данные сборка 001 не сохраняет.

## 5. Импорт Hysteria2

Повторите импорт со ссылкой `hysteria2://` или `hy2://`.

В карточке должен быть указан движок `sing-box`.

## 6. Импорт AmneziaWG

Вставьте конфигурацию с секциями:

```ini
[Interface]
...

[Peer]
...
```

В карточке должен быть указан движок `AmneziaWG`.

## 7. Разрешение VPN

Выберите профиль, вернитесь на экран «Подключение» и нажмите «Подготовить Android VPN».

Android должен показать стандартный системный запрос доверия к VPN-приложению. После подтверждения приложение показывает только факт получения разрешения, но не заявляет о работающем туннеле.

## 8. Журнал

Откройте «Журнал». Там должны быть отражены:

- запуск;
- импорт;
- выбор профиля;
- смена темы;
- результат запроса разрешения VPN.
